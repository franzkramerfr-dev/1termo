🐍 Curso: Lógica de Programação com Python
Repositório dedicado ao armazenamento de notas de aula, exemplos práticos de código e exercícios desenvolvidos durante o curso de Lógica de Programação.

📌 Conteúdo do Repositório
O curso está estruturado nos seguintes módulos de aprendizado:

Aula 01: Fundamentos de lógica, variáveis e funções de entrada/saída (input/print).
Aula 02: Estruturas de decisão condicionais (if, elif, else) e operadores lógicos.
Aula 03: Estruturas de repetição automáticas utilizando os loops while e for.
Aula 04: Manipulação de coleções de dados através de estruturas de Listas.
🛠️ Tecnologias Utilizadas
Linguagem: Python 3.x
Versionamento: Git e GitHub
Formatação: Markdown para documentação
🚀 Como Executar os Códigos
Certifique-se de ter o Python instalado em sua máquina.
Clone este repositório em seu terminal:
git clone github.com
Acesse a pasta do projeto:
cd seu-repositorio
Execute qualquer arquivo de script desejado:
python nome_do_arquivo.py
